Archivo Qué se hace

assets/css/main.css Estilos del mapa, puntos y modal
assets/js/main.js Lógica JS del modal (abrir, cerrar, keyboard)

Flujo de datos

Admin sube JPG del croquis → guardado en post meta eu_croquis_image
Admin agrega hotspots (nombre + imagen + X% + Y%) → guardado en eu_croquis_hotspots (JSON serializado)
single-eu_project.php lee los metas → renderiza imagen + círculos posicionados
Usuario hace clic en un punto → JS abre modal con imagen y nombre
Usuario cierra → clic fuera, botón ✕ o tecla Escape
